# 数据源获取指南

本程序使用 **MacCMS V10（苹果CMS）** 标准采集接口。

## 一、当前内置数据源

| 源名称 | API 地址 | 稳定性 | 备注 |
|--------|----------|--------|------|
| 光速资源 | `https://api.guangsuapi.com/api.php/provide/vod` | ⭐⭐⭐⭐⭐ | 推荐首选，链接质量好 |
| 量子资源 | `https://cj.lziapi.com/api.php/provide/vod` | ⭐⭐⭐ | 部分链接失效 |
| 暴风云资源 | `https://bfzyapi.com/api.php/provide/vod` | ⭐⭐ | 部分可用 |

## 二、如何寻找更多数据源

### 方法 1：搜索引擎搜索

搜索关键词：
```
苹果CMS V10 资源站
MacCMS 采集站 2025
影视资源站 API
免费视频采集接口
资源采集网
```

### 方法 2：常用域名模式

寻找符合以下模式的域名：

```
api.xxxapi.com/api.php/provide/vod
cj.xxxzy.com/api.php/provide/vod
caiji.xxx.com/api.php/provide/vod
www.xxxzy.com/api.php/provide/vod
```

### 方法 3：从影视导航站获取

很多影视聚合站会列出合作的采集源，查找：
- 标注有"苹果CMS"、"MacCMS"的站点
- 提供"采集 API"说明的站点

## 三、测试数据源是否有效

### 方式 1：curl 命令测试

```bash
# 测试搜索功能
curl "https://域名/api.php/provide/vod?wd=三体"

# 测试列表功能
curl "https://域名/api.php/provide/vod?ac=list&pg=1"
```

**判断标准**：
- 返回 `Content-Type: application/json`
- 包含 `list` 字段（剧集列表）
- 状态码 200

### 方式 2：浏览器直接访问

在浏览器地址栏输入：
```
https://域名/api.php/provide/vod?wd=三体
```

如果看到 JSON 格式数据就是有效的。

## 四、如何添加新数据源到程序

### 方式 1：修改源代码（推荐）

编辑 `ovd/config.py`：

```python
DEFAULT_SOURCES: tuple[Source, ...] = (
    Source(name="光速资源", api="https://api.guangsuapi.com/api.php/provide/vod"),
    Source(name="量子资源", api="https://cj.lziapi.com/api.php/provide/vod"),
    Source(name="暴风云资源", api="https://bfzyapi.com/api.php/provide/vod"),
    # 在这里添加新的数据源
    Source(name="新数据源名称", api="https://新域名/api.php/provide/vod"),
)
```

### 方式 2：等待程序更新

如果发现了好用的数据源，可以提交 Issue，我会在新版中内置。

## 五、质量判断标准

一个好的数据源应该满足：

1. ✅ **搜索响应快**：< 3 秒返回结果
2. ✅ **数据量大**：至少几万部影视资源
3. ✅ **链接有效**：m3u8 链接可用率 > 80%
4. ✅ **无反爬限制**：可以正常连续访问
5. ✅ **更新及时**：新剧上架速度快

## 六、注意事项

### ⚠️ 常见问题

1. **403 Forbidden**：站点有反爬，需要 UA 或限制 IP
2. **500/502/503**：服务器故障或维护中
3. **返回 HTML**：需要登录或被跳转，API 不可用
4. **JSON 格式错误**：非标准 MacCMS 接口

### ⚠️ 使用风险

1. **链接经常失效**：免费资源站的播放链接很容易过期
2. **反爬限制**：大量请求可能被封 IP
3. **广告/水印**：部分站点的视频内嵌广告或水印
4. **版权问题**：仅供学习研究使用，请支持正版

## 七、光速资源 API 示例

### 搜索接口

```
GET https://api.guangsuapi.com/api.php/provide/vod?wd=遮天
```

响应字段：
- `list`: 搜索结果列表
- `vod_name`: 剧名
- `vod_pic`: 封面图
- `vod_remarks`: 更新状态

### 详情接口

```
GET https://api.guangsuapi.com/api.php/provide/vod?ac=videolist&ids=12345
```

响应字段：
- `vod_play_from`: 播放源名称
- `vod_play_url`: 播放链接列表
